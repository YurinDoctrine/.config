# Install this config

```sh
curl --tlsv1.2 -fsSL https://raw.githubusercontent.com/YurinDoctrine/.config/artix/.local/bin/cbpp-welcome >cbpp-welcome && \
 chmod 0755 cbpp-welcome && \
 ./cbpp-welcome

```
